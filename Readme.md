## Проект: "Админ-панель для управления товарами на ExtJS"

**Frontend-разработчик**

**Цель проекта:**
Разработка SPA-приложения на ExtJS 6+ с системой авторизации и CRUD-функционалом для управления товарами.

**Используемые технологии:**

- ExtJS 7.9.0.20

**Запуск проекта**
```
sencha app watch
```
